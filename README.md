‚ùìWhat is "Easy Launcher"?

Easy Launcher is a Minecraft launcher written in Python.

‚ùìWhat do you need for Easy Launcher?

You will need two programs:
1) Python -- https://www.python.org/downloads/
2) Java -- https://www.java.com/

‚ùìHow do I install and play with Easy Launcher?

üñ•Installation:

1) Download Easy_Launcher.zip
2) Unzip it to a convenient location
3) Run Requirements_Installer.py
4) Edit minecraft_path.json, where "path" is the path to the Minecraft folder, and "path2" is the path to the Java folder

üñ•Launch:

1) Run Launcher.py
2) Select "Install Version" and enter the version number (example: 1.16.5)
3) Wait for the installation to complete, then select "Launch Minecraft," enter the downloaded version number, and enter your nickname.
4) Done! Now you can play Minecraft alone or with friends.
